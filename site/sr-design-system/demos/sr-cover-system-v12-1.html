v12 Moves Applied

1. Tab as hard object

Fixed 100px height across all tiers. Width is the only variable (3/14/18px). Hard bottom border creates physical cutoff. Gateway tab fades (opacity 0.6, no border) — paid tabs are crisp objects.

2. Marks committed

Fewer strokes, thicker weight (1.6–2.0), higher opacity (0.65). Each mark is stone-carvable: cycle (Renergence), load split (HNR), crosshair (Engagement), spectrum bars (MN). No decoration — geometric facts.

3. Typographic inevitability

Every title breaks structurally: "The / Bottleneck Trap" not "The Bottleneck / Trap." Subtitles break at clause boundaries. No line break could move without breaking meaning.

4. Tier tension

Gateways: 80% title opacity, 70% subtitle, faded 3px tab with no border. Paid: full opacity, labeled tab with hard edge. The visual weight difference makes gateways feel like doorways — you sense the gravity is elsewhere.

5. Amazon stress test

Tested at 60px (thumbnail), greyscale, and blur. Color fields remain distinct. Marks register as shapes. Tab reads as an object even when text is illegible. The system survives.

6. Signature constraint

Author name is always the quietest element: 5.5px, 55% opacity, maximum tracking. The system is louder than the person. This is the fingerprint — every SR cover subordinates authorship to architecture. No other publisher would choose this.

7. Gateway assertion (v12.1 fix)

Subtitle dropped to 45% opacity (from 70%). The title now stands alone harder on gateways. The door is visible; the hallway behind it is not. You walk through, not into.

8. MN irrational constraint (v12.1 fix)

MN spectrum bars are mathematically locked: x=10 start, endpoints at 42/28/38/20/34, Y at 9px intervals, stroke always 2.0. This grid is identical on every MN cover — only title and subtitle change. No other thread's marks are this rigid. The constraint is irrational (no one would choose it) and inevitable (once you see it, it could not be otherwise).
