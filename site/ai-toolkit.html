AI Toolkit

When you use AI with the Renergence books, the AI doesn't know how to interpret the frameworks. It defaults to generic advice. These toolkit files fix that.

The toolkit governs how AI behaves. The books provide what it works with. You need both.

Read the AI policy →

What It Does

Drop the toolkit file into your AI project alongside the books. The AI will:

  • Identify patterns by name from the books
  • Ask which domain applies before responding
  • Hold competing explanations instead of collapsing to one
  • Ground everything in the actual book content
  • Distinguish diagnosis from advice

Setup

Claude (Anthropic)

  1. Create a new Project
  2. Upload the toolkit file to the project knowledge
  3. Upload the book files (.md) to the same project
  4. Start a conversation—the toolkit applies automatically

Works on the free plan. Pro ($20/mo) gives unlimited file uploads and access to stronger models.

ChatGPT (OpenAI)

  1. Start a new conversation
  2. Upload the toolkit file and a book file
  3. The AI will use the toolkit for that conversation

Works on the free plan. For persistent use across conversations, create a Custom GPT (requires Plus at $20/mo) and add the files there.

Other AI Tools

  1. Open your AI tool (Gemini, Cursor, or similar)
  2. Upload or paste the toolkit file as project instructions
  3. Upload the relevant book file

Any tool that accepts file uploads or system prompts will work.

One More Thing

The toolkit makes AI smarter about the material. It doesn't make it a substitute for the material. The books do something AI can't—they create recognition through pacing, sequence, and the way they hold ambiguity before resolving it. That experience matters. Read them.

If you work with clients: this toolkit helps with preparation and reflection, but the real diagnostic work—the instruments, the routing, the calibration—that comes from practitioner training. AI is a good thinking partner. It's not a practitioner.
